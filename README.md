# 🔐 Cyber Security Behavioral Risk Analyzer

## Overview

Cyber Security Behavioral Risk Analyzer is a machine learning-based cybersecurity assessment system that evaluates user behavior, security habits, and decision-making patterns to identify cyber risks. The application analyzes responses to cybersecurity-related scenarios and predicts whether a user falls into a Low, Medium, or High risk category.

The project combines behavioral analytics, feature engineering, machine learning, and an interactive Streamlit dashboard to provide personalized security recommendations and risk insights.

## Installation

Clone the repository:

```bash
git clone https://github.com/kckcode/Cyber-Security-Behavioral-Risk-Analyzer.git
cd Cyber-Security-Behavioral-Risk-Analyzer
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
